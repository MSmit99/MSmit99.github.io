Foundations of AI and Data Science

This hands-on course introduces the interdisciplinary field of AI and data science, highlighting both technical and ethical perspectives. Students use Python tools to analyze data and build models such as regression, classification, clustering, and dimensionality reduction. The course also explores societal impacts and ethical issues, helping students develop technical skills while learning to apply AI responsibly in real-world contexts.

Big Data

This course explores techniques for collecting, storing, analyzing, and visualizing big data with a focus on scalable machine learning. Students learn the MapReduce paradigm using Hadoop and Pig, and use Apache Spark for distributed processing, machine learning, and streaming. Topics include batch and real-time analysis, clustering, regression, classification, SQL vs. NoSQL databases, and natural language processing applications.

Machine Learning

An overview of machine learning techniques using R. Topics include an introduction to programming in R, the use of nearest neighbor and naive Bayes algorithms, support vector machines, hidden Markov models, and market basket analysis as well as model evaluation and improvement.

Applications of Generative AI

This course introduces generative artificial intelligence for students with Python experience. It explores how generative models create synthetic data, produce content, and address real-world problems. Students also learn prompt engineering techniques for interacting effectively with AI systems. Through hands-on projects and discussions of ethics, the course builds practical skills for responsible use of generative AI.
