Application Development

This course provides an in-depth exploration of creating applications for Android and iOS platforms. Students gain proficiency in programming languages for mobile app development, engaging in hands-on projects to design, develop, and deploy functional mobile applications. The course balances theoretical knowledge with practical exercises, preparing students for careers in the dynamic field of mobile app development.

Web Development

A study of the architecture and functionality of the Internet and the World Wide Web is followed by an in-depth study of website design, including comprehensive coverage of HTML, XML, and other tools needed to establish an interactive website with a database back-end. Client-side programming will be done in JavaScript. Server-side programming will be done in a variety of languages that may include PHP, JSP, ASP.NET, or frameworks such as Rails.

Software Engineering

An introduction to software development methodologies including requirements specification, design, testing, maintenance, and documentation. Students will participate in a large team-based software development project using version control software.

Database Systems

An introduction to database concepts and database management systems, focusing on how data is stored, processed, evaluated, displayed, and secured. The course covers logical and physical database design, key management techniques, and common data languages such as SQL and QBE. Students explore relational and object-oriented development approaches and gain practical experience creating databases and information systems using widely used database software.
