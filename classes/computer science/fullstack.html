Application Development

This course provides an in-depth exploration of creating applications for Android and iOS platforms. Students gain proficiency in programming languages for mobile app development, engaging in hands-on projects to design, develop, and deploy functional mobile applications. The course balances theoretical knowledge with practical exercises, preparing students for careers in the dynamic field of mobile app development.

Web Development

Covers Internet and web architecture, followed by in-depth website design using HTML, XML, and related tools. Includes client-side JavaScript and server-side development in languages such as PHP, JSP, ASP.NET, or Rails, with interactive, database-backed web applications.

Software Engineering

Requirements, design, testing, documentation, and team-based project using version control.

Database Systems

SQL, data modeling, relational + NoSQL DBs, Oracle, MySQL, security, and data processing.
